# Snake Game

A classic Snake Game built with HTML, CSS, and JavaScript.

## Features
- Classic snake gameplay
- Score tracking
- Highest score persistence using localStorage
- Game timer
- Start and restart modals

## Controls
- Use **Arrow Keys** to move the snake

## Play the Game
Live demo: https://hungryy-snakee.netlify.app/

## How to Run Locally
1.  Clone or download this repository
2.  Open `index.html` in your browser
